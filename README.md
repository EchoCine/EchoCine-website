# EchoCine Website Preview

在线预览网站

## 如何部署到 GitHub Pages

1. 在GitHub创建新仓库，命名为 `echocine-website`
2. 上传所有文件到仓库
3. 进入 Settings > Pages 
4. Source 选择 "Deploy from a branch"
5. Branch 选择 "main"，文件夹选择 "/ (root)"
6. 保存后等待1-2分钟，访问 `https://你的用户名.github.io/echocine-website`

## 或者使用 Netlify Drop

1. 访问 https://app.netlify.com/drop
2. 把整个文件夹拖拽进去
3. 自动获得在线链接
